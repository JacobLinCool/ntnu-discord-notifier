# NTNU Discord Notifier

Discord Webhook Notifier.

## For Operators

Run `docker compose up -d` to start the service.

## For Developers

### Prerequisites

* Node 18+
* PNPM 7+

### Setup

```sh
pnpm i && pnpm build
```

### Dev-Run

```sh
pnpm dev
```

> If you want to add a notifier, please checkout the upstream library: <https://github.com/JacobLinCool/NTNU-Notifier>
